# IP Changer / Pengubah IP

## English Version

### Description
IP Changer is a cross-platform application that allows users to change their IP address automatically or manually using proxy servers.

### Requirements
- Python 3.6+
- PyQt5
- requests

### Installation
1. Clone this repository:
   ```
   git clone https://github.com/yourusername/ip-changer.git
   ```
2. Navigate to the project directory:
   ```
   cd ip-changer
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

### Usage
1. Run the application:
   ```
   sudo python ip_changerEN.py
   ```
2. Click "Update Proxy List" to fetch and verify working proxies.
3. Use "Change IP Once" to change your IP address manually.
4. Select an interval and click "Start Automatic IP Change" for automatic IP changes.
5. Click "Stop" to halt automatic IP changes.
6. Use "Reset IP" to revert to your original IP address.

### Notes
- Ensure you have the necessary permissions to change system proxy settings.
- The application works best with an active internet connection.
- Some antivirus software may interfere with proxy changes.

### Platforms
- Windows: Tested on Windows 10
- macOS: Tested on macOS Catalina and later
- Linux: Tested on Ubuntu 20.04 (may require additional setup for different desktop environments)

## Versi Bahasa Indonesia

### Deskripsi
Pengubah IP adalah aplikasi lintas platform yang memungkinkan pengguna untuk mengubah alamat IP mereka secara otomatis atau manual menggunakan server proxy.

### Persyaratan
- Python 3.6+
- PyQt5
- requests

### Instalasi
1. Klon repositori ini:
   ```
   git clone https://github.com/yourusername/ip-changer.git
   ```
2. Masuk ke direktori proyek:
   ```
   cd ip-changer
   ```
3. Instal dependensi yang diperlukan:
   ```
   pip install -r requirements.txt
   ```

### Penggunaan
1. Jalankan aplikasi:
   ```
   sudo python ip_changerID.py
   ```
2. Klik "Update Proxy List" untuk mengambil dan memverifikasi proxy yang berfungsi.
3. Gunakan "Change IP Once" untuk mengubah alamat IP Anda secara manual.
4. Pilih interval dan klik "Start Automatic IP Change" untuk perubahan IP otomatis.
5. Klik "Stop" untuk menghentikan perubahan IP otomatis.
6. Gunakan "Reset IP" untuk kembali ke alamat IP asli Anda.

### Catatan
- Pastikan Anda memiliki izin yang diperlukan untuk mengubah pengaturan proxy sistem.
- Aplikasi berfungsi paling baik dengan koneksi internet aktif.
- Beberapa perangkat lunak antivirus mungkin mengganggu perubahan proxy.

### Platform
- Windows: Diuji pada Windows 10
- macOS: Diuji pada macOS Catalina dan versi lebih baru
- Linux: Diuji pada Ubuntu 20.04 (mungkin memerlukan pengaturan tambahan untuk lingkungan desktop yang berbeda)
